**W4LLB4NG3D** is a W4LLH4CK for the game VALORANT.<br>
**W4LLB4NG3D** comes with features like ESPs, Tracers, Health bars and distance calculator.

## Features
- **ESP**
  - Multiple ESP modes
    - Box
    - Hp Bars
    - Full
    - Distance
    - Angle box
    - Spine Lines
  - Toggle Tracers
- **Customization**
  - Custom colors
    - Change Box color
    - Change Tracer color
